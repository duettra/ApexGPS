# دليل ApexGPS

تطبيق مشي مجاني للأندرويد. عرض مسارات GPX، تحديد نقاط، تنزيل خرائط للاستخدام دون اتصال، ومشاركة موقعك المباشر مع صديق. بلا حساب، بلا إعلانات، بلا سحابة.

يمرّ هذا الدليل على كل ما يمكن للتطبيق القيام به. ابدأ بـ **البدء** إذا ثبّتّه للتو؛ خلاف ذلك انتقل إلى القسم الذي تحتاجه.
