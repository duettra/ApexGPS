# Erste Schritte

## Installation

ApexGPS ist im Play Store verfügbar: [play.google.com/store/apps/details?id=com.apexgps.app](https://play.google.com/store/apps/details?id=com.apexgps.app). Direkt aus dem Eintrag installieren — keine Tester-Einladung mehr nötig.

## Erster Start

Wenn Sie die App öffnen, fragt sie nach der **Standort-Berechtigung**. Tippen Sie auf **Während der Nutzung der App** (oder **Genau**, falls das System genau und ungefähr trennt — ApexGPS braucht präzises GPS zum Wandern).

Sie landen auf der Karte. Der Standard-Kartenstil bei einer Neuinstallation ist **ESRI Topo** (saubere Beschriftungen bei Wander-Zoomstufen); bestehende Installationen behalten den zuletzt gewählten Stil. Sie können den Standard jederzeit unter **Einstellungen → Darstellung → Standardkarte** ändern oder den Stil mitten in der Sitzung über den Layer-Button wechseln.

Beim ersten Start einer Neuinstallation wird eine reale Aufzeichnung der **Watzmann-Überschreitung** (Berchtesgaden) automatisch als Beispiel importiert — eine ~22,6 km Rundtour mit drei markanten Wegpunkten, sodass die App nicht leer ist, wenn Sie sie öffnen. Wenn Sie die Beispiele löschen und zurückhaben möchten, importiert sie **Einstellungen → Daten → Beispieldaten laden** erneut.

## Fünf-Minuten-Rundgang

### Sehen, wo Sie sind

Tippen Sie auf den **Fadenkreuz-Button** in der rechten Spalte. Das schaltet GPS ein und zentriert die Karte auf Ihre Position. Ein blaues Dreieck erscheint an Ihrem Standort — es zeigt in Ihre Bewegungsrichtung, wenn Sie sich bewegen.

### Umsehen, ohne sich zu verlieren

Ziehen Sie die Karte zum Verschieben, pinchen Sie zum Zoomen, nutzen Sie zwei Finger zum Drehen. Das Dreieck aktualisiert sich weiter, während Sie sich bewegen, aber die Karte zieht nicht zu Ihnen zurück. Zum Re-Zentrieren tippen Sie erneut auf den Fadenkreuz-Button.

Um GPS ganz auszuschalten (und Akku zu sparen), **lange drücken** Sie den Fadenkreuz-Button.

### Einen Wegpunkt hinzufügen

Lange drücken auf irgendeinen Punkt der Karte. Ein Dialog öffnet sich — wählen Sie einen Namen, eine Farbe, eines von 32 Symbolen (Gipfel, Aussicht, Wasserfall, Trinkwasser, Rastplatz, usw.). **Speichern**. Ein kleiner farbiger Pin steht nun an diesem Ort.

### Einen GPX-Track importieren

Öffnen Sie **Menü → Tracks** → tippen Sie auf das **Ordner-Symbol** als FAB unten rechts → wählen Sie eine oder mehrere `.gpx`-Dateien. Die Tracks erscheinen auf der Karte; tippen Sie einen an, um Details zu sehen (Distanz, Höhenprofil).

### Eine Tour aufzeichnen

Tippen Sie auf den roten **Aufnahme-Punkt** oben links. Der Punkt erweitert sich zu einem Live-Timer; eine rote Breadcrumb zeichnet Ihren Pfad, während Sie sich bewegen. Tap auf den Timer für **Pause / Fertig / Löschen**. Bei **Fertig** wird die Aufzeichnung als neuer Track gespeichert. Siehe [Tracks → Aufzeichnung](tracks.md#aufzeichnung).

### Ihren Standort teilen

Tippen Sie auf das blaue Dreieck auf der Karte. Ein Panel fährt nach oben mit Höhe, Genauigkeit, Uhrzeit, Akkustand und zwei Kartenlinks. Tippen Sie **Teilen** → wählen Sie WhatsApp, SMS, E-Mail, was immer. Der Empfänger sieht die Infos + einen Link; wenn er auch ApexGPS installiert hat, öffnet der Link direkt die App.

### Alles teilen, was gerade auf Ihrem Bildschirm ist

Tippen Sie auf das **Teilen-Symbol** oben links auf der Karte (links vom ☰ Menü). Ein Sheet öffnet sich mit jedem sichtbaren Track + Wegpunkt in Ihrem Kartenausschnitt — haken Sie an, was Sie senden möchten, tippen Sie auf Teilen, wählen Sie den Messenger. Das ganze Bündel geht als einzelne `.gpx`-Datei raus. Siehe [Sichtbaren Bereich teilen](share-and-navigate.md#alles-teilen-was-gerade-sichtbar-ist-sichtbaren-bereich-teilen).

### Zurück zum Hauptmenü

Das **☰ Menü** (oben links) hat Tracks, Wegpunkte, Karten, Einstellungen. Von dort verwalten Sie alles, was Sie importiert oder heruntergeladen haben.
